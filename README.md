# Скрипт по удалению мусорных писем из почтового ящика превращаем его как-будто только что создал.
  
Для корректной работы потребуется адрес почтового ящика и после запуска скрипта потребуется ввести пароль по образцу ниже.
  
  Пароли как сид фраза из 4х комбинаций 4х-значных слов 

Например: 

      qwer asdf zxcv qsdv
